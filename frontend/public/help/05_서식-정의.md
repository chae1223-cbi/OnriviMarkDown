온리비 어서의 **CSS 프로필 시스템**은 문서 요소별로 글꼴, 크기, 색상, 여백, 정렬 등을 세밀하게 제어할 수 있는 강력한 서식 정의 도구입니다. 

## 프로필 개요

CSS 프로필은 문서의 시각적 스타일을 정의하는 단위입니다. 여러 개의 프로필을 만들고 상황에 따라 전환할 수 있습니다.

- **프로필 생성**: 설정 > 서식 정의 > 새 프로필
- **프로필 전환**: 프로필 드롭다운에서 다른 프로필 선택
- **프로필 복제**: 기존 프로필을 복제하여 변형 생성
- **프로필 삭제**: 더 이상 필요하지 않은 프로필 제거
- **프로필 가져오기/내보내기**: JSON 형식으로 프로필 공유

## 설정 가능한 항목

### 페이지 스타일

| 항목               | 설명                             |
| ------------------ | -------------------------------- |
| 글꼴               | 본문에 사용할 글꼴 패밀리        |
| 글자 크기          | 본문 기본 글자 크기              |
| 줄 간격            | 행간(LINE-HEIGHT)                |
| 자간               | 글자 간격(LETTER-SPACING)        |
| 배경색             | 문서 배경색                      |
| 방향               | 세로(portrait) / 가로(landscape) |
| 머리글 크기 오프셋 | h1~h6의 상대적 크기 조절         |

### 용지 여백 (mm)

PDF 내보내기 시 적용될 여백을 mm 단위로 설정합니다.

- 위/아래/왼쪽/오른쪽 각각 설정 가능
- 기본값: 각 10mm

### 요소별 스타일

다음 HTML 요소에 대해 개별적으로 스타일을 지정할 수 있습니다:

| 요소          | 설명                                      |
| ------------- | ----------------------------------------- |
| h1 ~ h6       | 제목 (정렬, 굵기, 여백)                   |
| p             | 본문 단락 (정렬, 여백, 들여쓰기)          |
| a             | 링크 (색상, 밑줄)                         |
| img           | 이미지 (정렬, 여백)                       |
| blockquote    | 인용문 (테두리, 여백, 배경색)             |
| table, th, td | 표 (테두리, 정렬, 셀 간격)                |
| code          | 인라인 코드 (배경색, 글자색, 둥근 모서리) |
| pre           | 코드 블록 (배경색, 테두리)                |
| li            | 목록 아이템 (여백)                        |
| hr            | 수평선 (스타일, 색상, 두께)               |

### 페이지 나누기

특정 요소 앞에서 페이지를 나누도록 설정할 수 있습니다:

- h1 ~ h6 각각에 대해 페이지 나누기 전/후 설정 가능
- 페이지 나누기 표시 (미리보기에서 빨간 점선)
