온리비 어서는 다양한 설정을 통해 사용자 경험을 커스터마이징할 수 있습니다. 

## 설정 메뉴 열기

- 상단 메뉴바의 **설정** 버튼(⚙️) 클릭
- 단축키: 설정 메뉴를 통해 접근

## 에디터 설정

| 설정             | 설명                | 기본값 |
| ---------------- | ------------------- | ------ |
| 자동 저장 간격   | 자동 저장 주기 (초) | 30초   |
| 자동 저장 활성화 | 자동 저장 켜기/끄기 | 켜짐   |
| 단어 잘림        | 긴 줄 자동 줄바꿈   | 꺼짐   |

## 테마 설정

| 설정                 | 설명                                            |
| -------------------- | ----------------------------------------------- |
| 테마                 | 라이트 / 다크 모드 전환                         |
| 시스템 테마 따라가기 | OS 설정과 동기화                                |
| 에디터 테마          | Monaco Editor 테마 (onrivi-light / onrivi-dark) |

### 에디터 테마 커스터마이징

에디터 테마는 HTML 미리보기와 별도로 Monaco Editor 자체의 색상 테마를 의미합니다. 다음 요소의 색상이 정의되어 있습니다:

| 토큰             | 설명                     |
| ---------------- | ------------------------ |
| `keyword.md`     | 마크다운 키워드 (볼드체) |
| `strong.md`      | 굵은 텍스트              |
| `emphasis.md`    | 기울임 텍스트            |
| `string.link.md` | 링크 문자열              |

## 보기 설정

| 설정                        | 설명                          | 기본값 |
| --------------------------- | ----------------------------- | ------ |
| 툴바 표시                   | 상단 서식 툴바                | 켜짐   |
| 사이드바 표시               | 좌측 파일 탐색기              | 켜짐   |
| 페이지 뷰                   | A4 페이지 형태 미리보기       | 꺼짐   |
| 내보내기 페이지 나누기 표시 | 페이지 구분선(빨간 점선) 표시 | 켜짐   |

## 단축키 사용자 정의

단축키 설정에서 각 명령어에 할당된 키보드 단축키를 변경할 수 있습니다.
<!-- [page-break] -->
### 변경 가능한 단축키

- 모든 서식 관련 단축키 (Bold, Italic, Heading 등)
- 파일 관련 단축키 (저장, 열기 등)
- 보기 관련 단축키 (툴바, 사이드바 등)

### 변경 방법

1. 설정 > 단축키 탭으로 이동
2. 변경하려는 명령어 옆의 키 입력 필드를 클릭
3. 원하는 단축키를 키보드에서 입력
4. 다른 단축키와 충돌이 있으면 경고 표시
5. 저장

## CSS 프로필

문서 서식 정의에 대한 자세한 내용은 [서식 정의 (CSS 프로필)](/help/05_서식-정의.md) 문서를 참고하세요.

## 라이선스

설정 > 라이선스에서 현재 라이선스 상태를 확인할 수 있습니다.

- 라이선스 키 입력
- 라이선스 상태 확인 (활성/만료/미등록)
- 라이선스 정보 표시
