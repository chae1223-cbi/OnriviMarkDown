온리비 어서는 작성한 문서를 네 가지 형식으로 내보낼 수 있습니다. 

## 내보내기 형식

| 형식 | 확장자 | 특징 |
|------|--------|------|
| PDF | .pdf | 인쇄 및 공유에 최적화, 페이지 레이아웃 유지 |
| HTML | .html | 웹에서 바로 열람 가능, 스타일 포함 |
| EPUB | .epub | 전자책 리더기 호환 |
| PNG | .png | 이미지 형태로 저장 (SNS 공유 등) |

## 공통 설정

내보내기 전에 설정할 수 있는 공통 옵션:

- **용지 방향**: 세로(Portrait) / 가로(Landscape)
- **용지 여백**: CSS 프로필에 설정된 여백값 적용
- **CSS 프로필**: 현재 선택된 CSS 프로필의 스타일이 내보내기에 반영
- **페이지 나누기**: CSS 프로필에서 설정한 페이지 나누기 규칙 적용

## PDF 내보내기

PDF 내보내기는 **html2canvas**로 미리보기 DOM을 캡처한 후 **jsPDF**로 PDF를 생성합니다.

### 처리 과정

1. 미리보기 DOM을 복제하고 이미지를 Base64로 인라인 변환
2. CSS 프로필 스타일을 복제본에 주입
3. html2canvas로 각 페이지 섹션을 고해상도(2x) 캡처
4. jsPDF로 A4 페이지에 배치
5. 페이지 배경색을 문서 배경색과 일치시켜 여백까지 동일한 색상으로 출력

### 특징

- 페이지 구분선을 기준으로 여러 페이지로 분할
- 콘텐츠가 한 페이지를 초과하면 자동으로 여러 페이지 생성
- 목록 마커(글머리 기호, 번호)가 html2canvas에서 누락되는 문제를 DOM 주입 방식으로 해결
- HTML `<br>` 태그가 PDF에서도 유지됨

## HTML 내보내기

HTML 내보내기는 Tailwind CSS Typography 플러그인이 포함된 단일 HTML 파일을 생성합니다.

- 모든 스타일이 인라인으로 포함되어 별도 CSS 파일 불필요
- 이미지는 Base64 Data URI로 인라인 변환
- 웹 브라우저에서 바로 열람 가능
- Google Fonts(Inter, Noto Sans KR) 링크 포함

## EPUB 내보내기

전자책 형식으로 내보내기 위해 EPUB 표준을 준수하는 파일을 생성합니다.

- EPUB 3.0 형식 준수
- CSS 프로필의 스타일이 EPUB 내부 스타일시트로 포함
- 이미지 Base64 인라인 변환 지원
- 전자책 리더기(리디, 교보, 알라딘 등)에서 호환

## PNG 내보내기

문서 전체를 단일 이미지로 캡처합니다.

- html-to-image 라이브러리 사용
- 고해상도(2x) 캡처
- 전체 문서 길이에 맞춰 자동 높이 조절
- CSS 프로필의 배경색 적용
- SNS 공유, 썸네일 등에 활용

## 저장 위치

- **Electron**: 시스템 파일 저장 다이얼로그를 통해 원하는 위치에 저장
- **브라우저**: 다운로드 폴더에 저장
- **Chrome 확장 프로그램**: 다운로드 폴더에 저장
