# 07_이미지 및 미디어 삽입 가이드

온리비 어서(Onrivi Author)는 단순한 텍스트 편집을 넘어 이미지, 유튜브 동영상 등의 시각 자료를 실시간으로 조화롭게 렌더링할 수 있는 강력한 미디어 통합 미리보기를 제공합니다. 

이 문서에서는 미리보기 영역에서 깨진 이미지(엑박) 없이 온전히 미디어를 불러오고 활용하는 올바른 삽입 방법을 안내합니다.

---

## 1. 이미지 삽입 기본 문법
마크다운에서 이미지를 불러올 때는 느낌표(`!`)로 시작하는 전용 태그 문법을 사용합니다.

```markdown
![이미지 설명](이미지_경로_또는_주소)
```
* **이미지 설명**: 이미지가 로드되지 않았을 때 대신 노출되는 텍스트이자 시각 장애인을 위한 대체 텍스트입니다. (생략 가능하나 `[]` 괄호는 유지해야 합니다.)
* **이미지 경로 또는 주소**: 이미지가 실제 존재하는 컴퓨터 하드디스크 상의 경로 또는 인터넷 URL 주소입니다.

---

## 2. 상황별 이미지 경로 지정 방식

### ① 웹 기반 이미지 (인터넷 주소)
웹 서버에 이미 업로드되어 있는 범용 이미지 주소는 별도의 변환 없이 원래 주소 그대로 삽입해 주면 미리보기에 즉시 나타납니다.
```markdown
![구글 로고](https://www.google.com/images/branding/googlelogo/1x/googlelogo_color_272x92dp.png)
```

### ② 로컬 상대 경로 이미지
현재 편집 중인 문서와 동일한 폴더 혹은 하위 폴더에 들어있는 이미지를 참조하는 기법입니다. 문서를 다른 컴퓨터로 옮기더라도 이미지 폴더 구조만 똑같이 유지하면 경로가 깨지지 않으므로 가장 권장되는 링크 방식입니다.
* **동일한 폴더의 이미지**: `![사진](photo.png)` 또는 `![사진](./photo.png)`
* **하위 폴더(images) 안의 이미지**: `![사진](images/photo.png)`
* **상위 폴더 안의 이미지**: `![사진](../photo.png)`

> [!TIP]
> 온리비 어서는 작성자가 상대 경로 이미지(`photo.png`)를 기입하면, 현재 작업 중인 마크다운 파일의 실제 물리 경로를 동적으로 역추적하여 온전하게 전체 절대 경로를 완성해 줍니다. 

### ③ 로컬 절대 경로 이미지 (`media://` 프로토콜)
로컬 하드디스크의 특정 저장 경로(예: `C:\MyFiles\picture.jpg`)를 직접 지정하는 방법입니다. 
데스크톱 브라우저 환경은 보안 샌드박스로 인해 로컬 디스크 파일 접근(`file:///`)을 엄격히 차단합니다. 

온리비 어서는 이 문제를 방지하기 위해 내장 미디어 스트리밍 서버 핸들러를 탑재하고 있습니다. 일반적인 경로명을 입력하더라도 프로그램이 내부적으로 **`media://` 가상 특수 프로토콜**을 적용해 보안 에러와 CORS 장벽을 우회하고 로컬 파일을 부드럽게 버퍼 스트림으로 로드하여 화면에 띄워줍니다.
```markdown
![내 사진](C:\Onrivi\Assets\banner.jpg)
```

## 3. 클립보드 이미지 복사 및 붙여넣기 (Ctrl + V)
인터넷에서 캡처하거나 그래픽 도구에서 복사한 이미지를 일일이 파일로 저장한 뒤 경로를 입력할 필요가 없습니다.

1. 웹브라우저나 캡처 프로그램에서 원하는 영역을 캡처하여 클립보드에 이미지 데이터를 담습니다.
2. 온리비 어서 편집 화면의 원하는 커서 위치에서 **`Ctrl + V`**를 누릅니다.
3. 프로그램이 자동으로 클립보드 이미지 데이터를 인지하여, 현재 편집 중인 파일의 동일 폴더(또는 지정 자산 폴더) 내에 고유 파일명(예: `image_1780112334840.png`)으로 물리 파일을 자동 생성 및 저장합니다.
4. 에디터 커서 자리에 저장된 이미지의 마크다운 링크 문법이 자동으로 자동 완성되어 삽입됩니다.

## 4. 유튜브(YouTube) 및 동영상 연동 방법
문서 내에 유튜브나 비디오 동영상을 내장하여 미리보기 창에서 재생이 가능하도록 할 수 있습니다.

### ① 유튜브 툴바 아이콘 사용
* 툴바의 비디오 모양 단추를 누르고 나타나는 입력창에 유튜브 비디오 URL(예: `https://www.youtube.com/watch?v=XXXXXX`)을 입력합니다.
* 에디터 창에 비디오용 마크다운 컨테이너가 자동으로 빌드됩니다.

### ② 수동 Iframe 코드 사용
마크다운 미리보기 창은 HTML 렌더링을 완전히 수용하므로, 유튜브 '공유 -> 퍼가기' 메뉴에서 제공하는 `<iframe>` 소스코드를 복사하여 문서 중간에 그대로 붙여넣어도 원활히 재생되는 동영상 창이 구현됩니다.
```html
<iframe width="560" height="315" src="https://www.youtube.com/embed/XXXXXX" frameborder="0" allowfullscreen></iframe>
```
