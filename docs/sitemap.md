# Onrivi Author 문서 사이트맵

> 전체 문서의 의미론적 구조와 관계를 보여줍니다.

```
docs/
├── README.md                     ← 전체 문서 인덱스 (이 파일)
├── sitemap.md                    ← 의미론적 개요 (현재 파일)
├── 00_WORKBENCH_RULES.md         ← [내부] Docu-Agent 워크벤치 규칙
├── 시작하기.md                    ← 시작하기 랜딩 페이지
└── 시작하기/                      ← 시작하기 하위 문서
    ├── 에디터-기본                ← Monaco Editor 편집 환경
    ├── 파일-관리                  ← 워크스페이스/파일 관리
    ├── 미리보기-모드              ← 뷰 모드/스크롤 동기화
    ├── 서식-정의                  ← CSS 프로필 시스템
    ├── 내보내기                   ← PDF/HTML/EPUB/PNG
    ├── 표-체크리스트              ← 표/체크리스트
    ├── 다이어그램-수식            ← Mermaid/KaTeX
    ├── 미디어-삽입                ← 이미지/YouTube/지도
    ├── 슬래시-명령어              ← 슬래시 명령어/단축키
    ├── 한글-입력                  ← IME/맞춤법/통계
    ├── ai-글쓰기                  ← AI 도우미
    ├── 설정                       ← 앱 설정
    └── 내보내기-고급              ← 고급 내보내기
```

## 문서 관계도

```
시작하기
├── 에디터 기본 사용법 ───────────────────── 슬래시 명령어 및 단축키
├── 문서 저장 및 파일 관리 ───────────────── 설정 및 커스터마이징
├── 미리보기 모드 ──────── 서식 정의 ──────── 내보내기 ──── 내보내기 고급
├── 표 및 체크리스트
├── 다이어그램 및 수식
├── 미디어 삽입
├── 한글 입력 및 글자 수 ─────────────────── AI 글쓰기 도우미
```

## 가이드

| 사용자 유형 | 추천 문서 |
|------------|-----------|
| 처음 시작하는 사용자 | 시작하기 → 에디터 기본 → 파일 관리 → 미리보기 모드 |
| 문서 서식을 원하는 사용자 | 서식 정의 → 내보내기 → 내보내기 고급 |
| 기술 문서 작성자 | 다이어그램 및 수식 → 표 및 체크리스트 |
| 한국어 사용자 | 한글 입력 → AI 글쓰기 |
| 앱 설정을 변경하려는 사용자 | 설정 → 슬래시 명령어 |
