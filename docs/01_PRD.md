# [PRD] Onrivi Author: 웹 기술 기반의 고성능 마크다운 지식 관리 시스템 (v2.0)

## 1. 프로젝트 개요
**Onrivi Author**는 개발자와 전문 작성자를 위한 **Next.js 기반의 고성능 마크다운 에디터**입니다. 로컬 파일 시스템과의 강력한 연동을 유지하면서도, 웹 기술의 유연함과 아름다운 디자인(Glassmorphism)을 결합하여 최상의 집필 환경을 제공합니다.

*   **핵심 가치:** 시각적 탁월함(Aesthetics), 로컬 우선(Local-First), 원활한 생산성(Seamless Productivity)

## 2. 핵심 아키텍처 및 기술 스택
*   **Frontend:** Next.js 14 (App Router), TypeScript, Tailwind CSS
*   **Editor:** Monaco Editor (VS Code 기반의 강력한 편집 성능)
*   **Backend:** Node.js (Express) - 로컬 파일 시스템(D:\워크스페이스) 제어 및 전역 검색 엔진
*   **Storage:** Browser File System Access API (브라우저 모드) & Local REST API (로컬 모드) 이원화 운영
*   **Key Libraries:** `react-markdown` (렌더링), `lucide-react` (아이콘), `html2pdf.js`/`html-to-image` (내보내기)

## 3. 주요 기능 요구사항

### 3.1 워크스페이스 및 파일 관리
*   **하이브리드 모드:**
    *   **로컬 모드:** 전용 백엔드를 통해 실제 로컬 디렉토리(`D:\워크스페이스`)와 실시간 동기화.
    *   **브라우저 모드:** 브라우저의 IndexedDB 및 File System Access API를 통한 샌드박스 저장.
*   **파일 탐색기:** 계층형 트리 구조(FileTree), 폴더/파일 생성·이름 변경·삭제 지원.
*   **목차(TOC) 생성:** 실시간 헤딩(H1~H3) 분석을 통한 문서 내 빠른 이동 지원.

### 3.2 고성능 편집 및 미리보기
*   **실시간 동기화:** 에디터와 미리보기 패널 간의 **양방향 스크롤 동기화** (Debounce 최적화).
*   **Monaco Editor 엔진:** 코드 하이라이팅, 다중 커서, 단축키 지원 등 VS Code 수준의 편집 경험.
*   **스마트 툴바:** 굵게, 기울임, 목록 변환, 체크리스트, 표/지도/이미지 삽입 모달 연동.

### 3.3 고급 도구 및 확장 기능
*   **전역 검색 (Global Search):** 백엔드 연동을 통해 전체 워크스페이스 내 파일명 및 **본문 텍스트 실시간 검색** 및 스니펫 강조.
*   **전문적 내보내기 (Export):**
    *   **HTML:** CSS가 포함된 독립 실행형 파일.
    *   **PDF:** 고품질 레이아웃 유지 (html2pdf).
    *   **PNG:** 전체 문서의 시각적 캡처 (html-to-image).
*   **특화 모달:** 지도(Map), 표 생성기(Table Generator), 이미지 삽입 로직 제공.
*   **다중 파일 병합(Merge):** 파일 탐색기 내 다중 선택을 통한 마크다운 파일 병합 제공. 구분선 삽입, 파일명 제목화, 소스 파일 자동 삭제 옵션 등 고급 통합 환경 지원.
*   **클립보드 이미지 물리 저장 및 삽입:** 드래그 앤 드롭 또는 붙여넣기를 통해 획득한 바이너리 이미지를 `assets/` 디렉토리에 실시간 영구 파일로 저장하고, 자동 마크다운 이미지 태그를 주입.
*   **원클릭 로컬 다운로드 저장:** PDF, HTML, PNG 등으로 내보낸 산출물을 시스템 다운로드 폴더(`Downloads`)에 백업 다이렉트 물리 파일화하여 저장.

### 3.4 시스템 안정성 및 백업
*   **에디터 백업 자동 복원 엔진:** 주요 프론트엔드 코드 파일(`page.tsx` 등)이 깨지거나 훼손되었을 때 VS Code/Cursor의 로컬 역사 백업 디렉토리를 자동 스캔하여 무결점 최신 사본을 추적하고 자동 복원/자가 치료하는 응급 복구 시스템 탑재.

## 4. UI/UX 디자인 명세
*   **Premium Dark/Light Mode:** 세련된 색상 팔레트와 글라스모피즘(Glassmorphism) 효과 적용.
*   **레이아웃 구성:**
    1.  **MenuBar:** 상단 전역 명령 및 설정 관리.
    2.  **Toolbar:** 편집 관련 퀵 액션 배치.
    3.  **Sidebar:** 탐색기(File)와 목차(TOC) 탭 전환 방식.
    4.  **Main Editor:** 5:5 분할 뷰 또는 프리뷰 전용 모드 전환.
    5.  **StatusBar:** 현재 작업 상태 및 폰트 크기 조절 기능.

## 5. 상세 기능 리스트 (Menu/Toolbar)
| 분류 | 기능 | 설명 |

| **파일(File)** | 새 파일, 열기, 저장, 내보내기 | 로컬/브라우저 저장소 연동 및 3종 내보내기 |
