## 2. 화면별 상세 정의

### 2.1 [SCR-01] 메인 작업장 (Main Workspace)
*   **컴포넌트 구성:** `MenuBar`, `Toolbar`, `FileTreeItem`, `Editor`, `Preview`, `StatusBar`
*   **세부 영역:**
    *   **파일 탐색기 탭:** 트리 구조 파일 관리, 우클릭/호버 액션(생성, 삭제, 이름 변경). **드래그 앤 드롭**을 이용한 파일/폴더 이동 및 최상단 헤더로의 루트 이동 지원.
    *   **목차(TOC) 탭:** 실시간 헤더 추출 및 클릭 시 해당 라인 이동.
    *   **편집 영역:** Monaco Editor 탑재, 라인 넘버링, 미니맵, 하이라이팅.
    *   **미리보기 영역:** 마크다운 실시간 렌더링, 수식 및 표 표시.
    *   **상태바:** 실시간 글자 수 계산, 폰트 크기 슬라이더, 드라이브 상태 표시.

### 2.2 [SCR-02] 전역 검색 (Global Search)
*   **기능:** 백엔드 API와 통신하여 전체 `.md` 파일 대상 키워드 검색.
*   **UI 요소:** 검색 입력창, 대소문자/정규식 토글, 결과 리스트(파일명+내용 스니펫).
*   **동작:** 검색 결과 더블 클릭 시 메인 에디터에 해당 파일 로드 및 검색어 위치 강조.

### 2.3 [SCR-03] 환경 설정 (Settings Modal)
*   **설정 항목:** 폰트 크기(Slider), 테마(Dark/Light), 줄 바꿈(On/Off), 자동 저장(On/Off), 드라이브 문자(D: 등), 워크스페이스 모드(Local/Cloud/Browser).
*   **특징:** 설정 변경 즉시 에디터와 UI에 반영되는 실시간 동기화 방식.

### 2.4 [SCR-04] 내보내기 도구 (Export Modal)
*   **옵션:** 
    *   **HTML:** 전체 스타일 포함 독립 파일.
    *   **PDF:** A4 규격 최적화 고해상도 문서.
    *   **PNG:** 전체 문서 수직 캡처 이미지.
*   **동작:** 변환 엔진(html2pdf, html-to-image) 구동 및 브라우저 다운로드 실행.

### 2.5 [SCR-05] 이미지 삽입 (Image Modal)
*   **기능:** 마크다운 이미지 구문(`![]()`) 생성 보조.
*   **필드:** 이미지 URL/경로 입력, 대체 텍스트(Alt text) 입력.
*   **결과:** 에디터의 현재 커서 위치에 마크다운 코드 즉시 삽입.

### 2.6 [SCR-06] 표 생성기 (Table Modal)
*   **기능:** 복잡한 마크다운 표 작성을 위한 그리드 도구.
*   **필드:** 행(Row) 개수 설정, 열(Column) 개수 설정.
*   **결과:** 지정된 행렬 크기의 마크다운 표 템플릿 자동 생성 및 삽입.

### 2.7 [SCR-07] 지도 삽입 (Map Modal)
*   **기능:** 구글/카카오 지도 위치 정보 공유.
*   **동작:** 위치 검색 또는 좌표 기반의 iframe 임베드 코드 생성 및 문서 삽입.

### 2.8 [SCR-08] 문법 가이드 (Markdown Helper)
*   **콘텐츠:** 자주 사용하는 마크다운 문법 예시 및 설명 리스트.
*   **동작:** 가이드 항목 클릭 시 해당 문법 템플릿이 에디터에 즉시 자동 삽입.

### 2.9 [SCR-09] 사용자 입력 (Prompt Modal)
*   **용도:** 파일/폴더 이름 변경, 새 항목 생성 시 텍스트 입력창 제공.
*   **UI:** 안내 제목, 기본값 입력 필드, 확인/취소 버튼.

### 2.10 [SCR-10] 알림 시스템 (Toast Notification)
*   **유형:** 성공(Success), 에러(Error), 정보(Info).
*   **동작:** 저장 성공, 내보내기 시작, 네트워크 에러 등의 실시간 상태 피드백 제공.

### 2.11 [SCR-11] 수식 에디터 (Formula Modal)
*   **기능:** LaTeX 문법을 이용한 전문 수식 작성 보조.
*   **주요 요소:**
    *   **도구 모음:** 그리스 문자, 연산자, 화살표 등 기호 선택 및 분수, 적분 등 템플릿 제공.
    *   **입력창:** LaTeX 소스 코드 직접 편집.
    *   **미리보기:** 입력 즉시 KaTeX 엔진을 통한 결과물 가시화.
*   **동작:** '인라인($)' 또는 '블록($$)' 모드를 선택하여 문서에 삽입.
