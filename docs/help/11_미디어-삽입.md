온리비 어서는 다양한 미디어를 문서에 삽입할 수 있는 기능을 제공합니다. [[시작하기로 가기]](/docs/help/00_시작하기.md)

## 이미지 삽입

### 기본 마크다운 이미지

```markdown
![대체 텍스트](이미지-경로)
```

### 지원하는 이미지 경로

| 유형      | 예시                            | 설명                  |
| --------- | ------------------------------- | --------------------- |
| 상대 경로 | `./images/photo.png`            | 문서 기준 상대 경로   |
| 절대 경로 | `/Users/me/photo.png`           | 파일 시스템 절대 경로 |
| 웹 URL    | `https://example.com/photo.jpg` | 외부 이미지 링크      |
| Data URI  | `data:image/png;base64,...`     | 인라인 Base64 이미지  |

### 이미지 크기 조정

URL 쿼리 파라미터로 이미지 크기를 조정할 수 있습니다:

```markdown
![이미지](image.png?width=300px)
![이미지](image.png?width=400px&height=300px)
```

- `width`, `w`: 이미지 최대 너비
- `height`, `h`: 이미지 높이

### 클립보드에서 이미지 붙여넣기

클립보드에 이미지가 있을 때 에디터에서 `Ctrl+V`를 붙여넣으면:

1. 이미지가 워크스페이스에 저장됩니다
2. `![이미지](저장된-경로)` 형식으로 자동 삽입됩니다
3. Electron 환경에서는 IPC를 통해 로컬 파일로 저장됩니다
4. Chrome 확장 프로그램 환경에서는 IndexedDB에 저장됩니다

## YouTube 동영상

슬래시 명령어 `/youtube`를 사용하여 YouTube 동영상을 삽입할 수 있습니다.

### 삽입 형식

미리보기에서는 YouTube 썸네일과 재생 버튼이 표시되며, 클릭하면 YouTube 페이지로 이동합니다.

내보내기(PDF/HTML) 시에는 썸네일 이미지와 하이퍼링크로 자동 변환됩니다.

## 지도 (Map)

슬래시 명령어 `/map`을 사용하여 지도를 삽입할 수 있습니다.

### 지원 지도 API

| API | 설명 |
|------|------|
| Yandex Maps | 정적 지도 이미지 렌더링 |
| Google Maps | Google Maps API 키 설정 필요 |

### API 키 설정

Google Maps API 키가 설정되지 않은 경우 자동으로 Yandex Maps로 폴백됩니다.

내보내기 시에는 정적 지도 이미지로 변환되어 포함됩니다.
