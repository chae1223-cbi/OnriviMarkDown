온리비 어서는 **Monaco Editor**(VS Code의 핵심 엔진)를 기반으로 한 마크다운 전용 에디터를 제공합니다.

## 기본 편집

- **WYSIWYG 스타일 미리보기**: 좌측에서 마크다운을 작성하면 우측에서 실시간으로 렌더링 결과를 확인할 수 있습니다.
- **구문 강조(Syntax Highlighting)**: 마크다운 문법(제목, 굵게, 코드 블록 등)이 에디터에서 색상으로 구분되어 표시됩니다.
- **자동 완성**: 마크다운 링크, 이미지 경로, 슬래시 명령어 등을 자동 완성할 수 있습니다.

## 단축키

| 단축키              | 동작                   |
| ------------------- | ---------------------- |
| `Ctrl+S`            | 저장                   |
| `Ctrl+Z` / `Ctrl+Y` | 실행 취소 / 다시 실행  |
| `Ctrl+F`            | 찾기                   |
| `Ctrl+H`            | 바꾸기                 |
| `Ctrl+Shift+S`      | 다른 이름으로 저장     |
| `Tab` / `Shift+Tab` | 들여쓰기 / 내어쓰기    |
| `Ctrl+B`            | 굵게 (`**텍스트**`)    |
| `Ctrl+I`            | 기울임 (`*텍스트*`)    |
| `Ctrl+U`            | 밑줄 (`<u>텍스트</u>`) |

> 전체 단축키 목록은 [슬래시 명령어 및 단축키](/docs/시작하기/슬래시-명령어) 문서를 참고하세요.

## 슬래시 명령어

에디터에서 `/` 키를 입력하거나 `/`입력후 'Ctrl + Enter'를 키를 입력하면 슬래시 명령어 자동완성 목록이 나타납니다.

### 지원 명령어 예시
/clean
| 명령어        | 설명                 |
| ------------- | -------------------- |
| `/bold`       | 굵은 텍스트 삽입     |
| `/italic`     | 기울임 텍스트 삽입   |
| `/h1` ~ `/h6` | 제목 삽입            |
| `/link`       | 링크 삽입            |
| `/image`      | 이미지 삽입          |
| `/table`      | 표 삽입              |
| `/inlincode`  | 인라인 코드          |
| `/code`       | 코드 블록 삽입       |
| `/quote`      | 인용문 삽입          |
| `/ul`         | 순서 없는 목록       |
| `/ol`         | 순서 있는 목록       |
| `/check`      | 체크리스트           |
| `/hr`         | 수평선 삽입          |
| `/math`       | 수식(KaTeX) 삽입     |
| `/map`        | 지도 삽입            |
| `/youtube`    | YouTube 동영상 삽입  |
| `/latex`      | LaTeX 수식 삽입      |
| `/now`        | 현재 날짜/시간 삽입  |
| `/cleandoc`   | 문서 정리(Clean Doc) |

## 들여쓰기 인텔리전스

마크다운 목록에서 `Tab` 키를 누르면 문맥을 인식하여 자동으로 2칸 들여쓰기가 적용됩니다. 빈 줄에서 `Tab`을 누르면 들여쓰기가 해제되고 목록이 일반 텍스트로 전환됩니다.

## Shift+Enter 줄바꿈

`Shift+Enter`를 누르면 표 내부에서 `<br>` 태그가 삽입되어 표 셀 내에서 줄바꿈을 할 수 있습니다.
