온리비 어서는 GFM(GitHub Flavored Markdown) 표와 체크리스트를 완벽하게 지원합니다.

## 표 (Table)

### GFM 표 작성

마크다운의 기본 표 문법을 사용합니다:

```markdown
| 헤더1 | 헤더2 | 헤더3 |
|-------|-------|-------|
| 내용1 | 내용2 | 내용3 |
| 내용4 | 내용5 | 내용6 |
```

### 향상된 표 기능

#### HTML/TSV에서 붙여넣기

- 엑셀, 구글 시트, Notion 등에서 복사한 표를 에디터에 붙여넣으면 자동으로 마크다운 표 형식으로 변환됩니다
- TSV(Tab Separated Values) 형식 지원
- HTML 테이블 형식 지원

#### 클립보드로 시트 복사

미리보기에서 표를 우클릭하면 클립보드에 TSV 형식으로 복사할 수 있습니다.

#### 표 정렬

헤더 구분선에 콜론(`:`)을 사용하여 열 정렬을 지정할 수 있습니다:

```markdown
| 왼쪽 정렬 | 가운데 정렬 | 오른쪽 정렬 |
| :-------- | :---------: | ----------: |
| 내용      | 내용        | 내용        |
```

### 표 삽입 명령어

`/table` 슬래시 명령어를 사용하면 3x3 기본 표가 삽입됩니다. 이후 행/열을 수동으로 추가할 수 있습니다.

## 체크리스트 (Checklist)

### 마크다운 체크리스트 문법

```markdown
- [ ] 완료되지 않은 항목
- [x] 완료된 항목
```

### 미리보기에서 체크리스트 클릭

미리보기에서 체크리스트를 클릭하면:

1. 체크박스 상태가 토글됩니다 (미완료 ↔ 완료)
2. 에디터의 해당 줄이 자동으로 업데이트됩니다
3. `[ ]` ↔ `[x]` 상태가 마크다운 소스에 반영됩니다

### 데이터 속성

미리보기의 각 요소에는 `data-line` 속성이 자동으로 부여되어, 클릭 시 에디터의 정확한 위치를 찾을 수 있습니다.
