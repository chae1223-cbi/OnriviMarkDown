온리비 어서는 한국어 사용자를 위한 다양한 기능을 제공합니다.[[시작하기로 가기]](/docs/help/00_시작하기.md)

## IME 조합 보호

한글은 자음과 모음이 조합되어 완성되는 IME(Input Method Editor) 기반 언어입니다. 온리비 어서는 IME 조합 중에 발생할 수 있는 문제를 방지합니다.

### 보호되는 동작

IME 조합 중(글자가 완성되기 전)에는 다음 동작이 차단됩니다:

- **슬래시 명령어 트리거**: `/` 키 입력이 IME 조합 중일 때는 슬래시 명령어가 실행되지 않습니다
- **Tab 키 들여쓰기**: IME 조합 중 Tab은 기본 들여쓰기만 수행하고 리스트 컨텍스트 처리는 건너뜁니다
- **Enter 키 리스트 처리**: IME 조합 중 Enter는 단순 개행만 수행하고 리스트 자동완성을 하지 않습니다
- **글로벌 단축키**: 모든 글로벌 키보드 단축키가 IME 조합 중에는 차단됩니다

이 보호 기능은 `e.isComposing` 이벤트 속성과 `__monaco_ime_composing` 플래그를 통해 구현됩니다.

### 내보내기 시 IME 커밋

PDF/PNG 내보내기 직전에는 강제로 IME 버퍼를 커밋(flush)하여 조합 중인 한글이 캡처되지 않도록 합니다.

## 글자 수 통계

<!-- [page-break] -->
상태 표시줄(Status Bar)에서 현재 문서의 통계를 확인할 수 있습니다:

- **글자 수**: 공백 포함/미포함 글자 수
- **단어 수**: 공백으로 구분된 단어 개수
- **원고지 매수**: 400자/200자 기준 원고지 매수

## Status Bar 정보

하단 상태 표시줄에는 다음 정보가 표시됩니다:

| 항목 | 설명 |
|------|------|
| 파일 이름 | 현재 편집 중인 파일명 |
| 폴더 | 현재 워크스페이스 폴더명 |
| 저장 상태 | 저장됨 / 저장되지 않음 |
| 커서 위치 | 현재 줄(line)과 열(column) 번호 |
| 글자 수 | 문서 전체 글자 수 |
| 워크스페이스 타입 | local / cloud / extension |
