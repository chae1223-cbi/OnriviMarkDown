온리비 어서는 `/` 키로 시작하는 슬래시 명령어와 다양한 키보드 단축키를 제공합니다.

## 슬래시 명령어

에디터에서 `/`를 입력하면 자동완성 목록이 나타납니다. `/` 입력 후 목록이 나타나지 않으면 `Ctrl + Space`를 입력하여 다시 불러올 수 있습니다.

### 텍스트 스타일

| 명령어        | 설명          | 삽입 결과      |
| ------------- | ------------- | -------------- |
| `/bold`       | 굵은 텍스트   | `**텍스트**`   |
| `/italic`     | 기울임 텍스트 | `*텍스트*`     |
| `/strike`     | 취소선        | `~~텍스트~~`   |
| `/inlinecode` | 인라인 코드   | `` `텍스트` `` |

### 제목

| 명령어        | 설명               |
| ------------- | ------------------ |
| `/h1` ~ `/h6` | 1단계 ~ 6단계 제목 |

### 목록

| 명령어   | 설명                  |
| -------- | --------------------- |
| `/ul`    | 글머리 기호 (`-`)     |
| `/ol`    | 숫자 목록 (`1.`)      |
| `/quote` | 인용구 (`>`)          |
| `/check` | 체크리스트 (`- [ ]`)  |

### 미디어

| 명령어   | 설명            |
| -------- | --------------- |
| `/link`  | 하이퍼링크 삽입 |
| `/image` | 이미지 삽입     |
| `/youtube` | 유튜브 동영상 삽입 |
| `/map`   | 지도 삽입       |

### 코드

| 명령어 | 설명                    |
| ------ | ----------------------- |
| `/code` | 코드 블록 (``` ```)     |
| `/table` | 표 삽입                 |
| `/math` | KaTeX 수식 (인라인: `$...$`) |

### 문서

| 명령어      | 설명                              |
| ----------- | --------------------------------- |
| `/hr`       | 수평선 (`---`)                    |
| `/now`      | 현재 날짜/시간 삽입               |
| `/cleandoc` | 문서 정리 (불필요한 공백 제거 등) |

## 키보드 단축키

### 파일

| 단축키         | 동작               |
| -------------- | ------------------ |
| `Ctrl+N`       | 새 파일            |
| `Ctrl+Shift+O` | 파일 열기          |
| `Ctrl+S`       | 저장               |
| `Ctrl+Shift+S` | 다른 이름으로 저장 |
| `Ctrl+W`       | 파일 닫기          |

### 편집

| 단축키 | 동작 |
|--------|------|
| `Ctrl+Z` | 실행 취소 |
| `Ctrl+Y` | 다시 실행 |
| `Ctrl+X` | 잘라내기 |
| `Ctrl+C` | 복사 |
| `Ctrl+V` | 붙여넣기 |
| `Ctrl+A` | 전체 선택 |
| `Ctrl+D` | 현재 줄 삭제 |

### 검색

| 단축키 | 동작 |
|--------|------|
| `Ctrl+F` | 찾기 |
| `Ctrl+H` | 바꾸기 |
| `Ctrl+Shift+F` | 전체 검색 |

### 텍스트 서식

| 단축키 | 동작 |
|--------|------|
| `Ctrl+B` | 굵게 |
| `Ctrl+I` | 기울임 |
| `Ctrl+K` | 링크 삽입 |

### 보기

| 단축키         | 동작          |
| -------------- | ------------- |
| `Ctrl+Shift+P` | 명령 팔레트   |
| `Ctrl+Shift+L` | 툴바 토글     |
| `Ctrl+Shift+B` | 사이드바 토글 |
| `Ctrl+Shift+M` | 모드 전환     |
| `Ctrl+=`       | 확대          |
| `Ctrl+-`       | 축소          |
| `Ctrl+0`       | 확대 초기화   |

### 기타

| 단축키 | 동작 |
|--------|------|
| `F11` | 전체 화면 |
| `Tab` | 들여쓰기 (목록 컨텍스트 인식) |
| `Shift+Enter` | `<br>` 줄바꿈 삽입 (표 내부) |

## 툴바

상단 툴바에서 주요 기능에 접근할 수 있습니다. 툴바는 `Ctrl+Shift+L`로 표시/숨김을 전환할 수 있습니다.

### 툴바 그룹

| 그룹     | 기능                                                                                       |
| -------- | ------------------------------------------------------------------------------------------ |
| 제목     | h1 ~ h6                                                                                    |
| 서식     | 굵게, 기울임, 취소선, 인라인 코드                                                          |
| 목록     | 순서 목록, 비순서 목록, 인용문, 체크리스트                                                 |
| 미디어   | 링크, 이미지, 동영상, 지도                                                                 |
| 코드     | 코드 블록, 표, 수식                                                                        |
| 문서     | 수평선, 날짜, 문서 정리                                                                    |
| 부가기능 | 왼쪽사이드바 보이기/숨기기, 툴바 보이기/숨기기, 모드 전환(미리보기/분할/편집)           |
요청하신 메뉴바 구조를 요청하신 단축키 및 툴바 목록과 동일한 형식의 표로 정리해 드립니다.
