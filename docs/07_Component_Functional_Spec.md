# [Component Spec] Onrivi Author: 컴포넌트별 상세 기능 정의서 (v1.0)

본 문서는 `frontend/src/components` 내의 각 컴포넌트가 수행하는 세부 기능과 데이터 입출력(Props/Events)을 정의합니다.

---

## 1. 전역 및 레이아웃 컴포넌트

### 1.1 `MenuBar.tsx`
*   **역할:** 최상단 시스템 메뉴바 제공 및 전역 명령 트리거.
*   **주요 기능:**
    *   파일, 편집, 보기, 도구 메뉴 드롭다운 관리.
    *   다크/라이트 모드 실시간 전환 스위치.
    *   사이드바 및 전역 검색창 토글 상태 제어.
*   **주요 인터페이스 (Props):**
    *   `isDarkMode`, `setIsDarkMode`: 테마 상태 및 변경 함수.
    *   `handlers`: `page.tsx`에서 정의된 각종 에디터 조작 함수 집합.

### 1.2 `Toolbar.tsx`
*   **역할:** 에디터 상단에서 자주 사용하는 마크다운 서식 및 모달 호출 버튼 배치.
*   **주요 기능:**
    *   서식(Bold, Italic, H1~H6 등) 즉시 적용 버튼.
    *   특수 객체(표, 지도, 이미지) 모달 오픈 트리거.
    *   폰트 크기 조절 및 자동 저장 상태 표시.
*   **주요 인터페이스 (Props):**
    *   `previewMode`: 'both' / 'preview' 모드 상태 제어.
    *   `handlers`: 서식 적용 및 모달 오픈 함수들.

### 1.3 `StatusBar.tsx`
*   **역할:** 화면 하단에 현재 문서의 메타데이터 및 환경 제어기 표시.
*   **주요 기능:**
    *   실시간 글자 수 및 단어 수 계산 표시.
    *   폰트 크기 슬라이더를 통한 에디터 가시성 조절.
    *   현재 작업 중인 파일의 모드(Local/Browser) 상태 인디케이터.

## 2. 데이터 및 워크스페이스 관리 컴포넌트

### 2.1 `FileTreeItem.tsx` (재귀 컴포넌트)
*   **역할:** 워크스페이스의 계층 구조를 렌더링하고 파일/폴더 CRUD 및 이동 수행.
*   **주요 기능:**
    *   폴더 접기/펴기(Toggle) 및 파일 더블 클릭 시 열기 연동.
    *   호버 시 노출되는 '추가', '이름 변경', '삭제' 버튼 기능.
    *   **드래그 앤 드롭 이동:** `onDragStart`로 소스 경로 추출, `onDrop` 시 타겟 경로로 `rename` API 호출.
    *   확장자별 아이콘 매칭 렌더링 (`helper.tsx` 연동).
*   **주요 인터페이스 (Props):**
    *   `node`: 현재 노드 데이터 (이름, 타입, 경로 등).
    *   `refreshParent`: 변경 사항 발생 시 부모 리스트 갱신 함수.

### 2.2 `GlobalSearch.tsx`
*   **역할:** 백엔드 API를 이용한 워크스페이스 내 전역 텍스트 검색.
*   **주요 기능:**
    *   디바운스(300ms)가 적용된 실시간 비동기 검색 요청.
    *   검색어가 포함된 텍스트 강조(Highlight) 처리 및 스니펫 표시.
    *   결과 항목 클릭 시 해당 파일 및 위치로 이동.

## 3. 기능형 모달 및 팝업 컴포넌트

### 3.1 `ExportModal.tsx`
*   **역할:** 문서를 외부 포맷으로 변환하기 위한 설정 팝업.
*   **주요 기능:**
    *   HTML, PDF, PNG 변환 엔진 호출 로직.
    *   변환 중 로딩 상태 표시 및 완료 시 브라우저 다운로드 실행.

### 3.2 `TableModal.tsx`
*   **역할:** 행/열 입력을 통한 마크다운 표 생성기.
*   **주요 기능:**
    *   사용자가 지정한 Row/Col 크기에 맞는 마크다운 그리드 문자열 생성.
    *   생성된 문자열을 에디터의 현재 커서 위치에 자동 삽입.

### 3.3 `FormulaModal.tsx`
*   **역할:** LaTeX 문법을 이용한 전문 수식 에디터.
*   **주요 기능:**
    *   템플릿 및 기호(그리스 문자 등)를 통한 LaTeX 소스 코드 생성.
    *   KaTeX 엔진을 활용한 실시간 결과물 미리보기.
    *   인라인($) 및 블록($$) 삽입 모드 지원.

### 3.4 `PromptModal.tsx` (공용)
*   **역할:** 시스템의 모든 텍스트 입력을 담당하는 범용 모달.
*   **주요 기능:**
    *   파일명, 폴더명 등 간단한 문자열 입력 필드 제공.
    *   `onConfirm`, `onCancel` 콜백을 통한 상위 컴포넌트 제어.

## 4. 유틸리티 및 알림 컴포넌트

### 4.1 `ToastProvider.tsx`
*   **역할:** 애플리케이션 전역의 알림 메시지 큐 관리.
*   **주요 기능:**
    *   `showToast` 함수를 통한 메시지 팝업 트리거.
    *   일정 시간 후 자동으로 사라지는 애니메이션 처리.

### 4.2 `ColorText.tsx` / `CopyButton.tsx`
*   **역할:** 텍스트 스타일링 및 클립보드 복사 보조.
*   **주요 기능:**
    *   특수 강조 텍스트 렌더링 및 클릭 시 즉시 복사 기능 수행.
